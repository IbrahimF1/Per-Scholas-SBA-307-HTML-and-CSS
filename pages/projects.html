Major Contributions

The Linux Kernel

Started in 1991 as a personal hobby, Linux has grown to become the most widely used operating system kernel in the world. It powers everything from the world's top supercomputers to the Android phone in your pocket.

Git

Created in 2005 out of frustration with existing proprietary tools, Git is a distributed version control system. It was designed to handle the massive scope of the Linux kernel development with speed and data integrity.

Subsurface

A passion project born from my love for scuba diving. Subsurface is an open-source dive log program that runs on Windows, Mac, and Linux. It allows divers to download data from their dive computers.
